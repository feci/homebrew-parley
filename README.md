# Homebrew Parley Tap

Homebrew tap for Parley Deck tools.

## Install

```bash
brew install feci/parley/parley-deck-cli
brew install feci/parley/parley-deck-skill
parley version
parley-deck-skill install
```

## Update

```bash
brew update
brew upgrade parley-deck-cli
brew upgrade parley-deck-skill
parley-deck-skill install --target all --force
```

## Doctor

```bash
parley-deck-skill doctor --target all
```
